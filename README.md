# 🇮🇳 BharatVerse: Rediscovering India, Digitally

**BharatVerse** is a digital platform made to support and promote India’s rich culture. It brings together traditional art, food, and languages in one place using modern technology.

---

## ✨ Features

- 🛍 **State-Wise Marketplace**  
  Explore and buy authentic handicrafts and textiles from different states. Each product includes the artisan’s story.

- 📚 **Learning Hub**  
  Learn through tutorials, videos, language flashcards, and cooking demos.

- 🎉 **Virtual Festivals**  
  Experience festivals like Pongal, Bihu, Onam, and Navratri online.

- 🏆 **Gamified Learning**  
  Earn points and badges by completing quizzes and activities.

- 🔐 **Secure System**  
  Safe login/signup with authentication and subscription plans.

---

## 🛠 Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB, Supabase  
- **Authentication:** Supabase Auth, JWT, bcrypt  
- **Payments:** Razorpay / Stripe  

---

## 📅 Roadmap

- **Phase 1:** Basic version (Login + UI)  
- **Phase 2:** Add features (Learning Hub + Dynamic Content)  
- **Phase 3:** Advanced (Mobile App + AI + AR/VR)
